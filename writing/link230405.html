23/04/05 一些新玩具和自己动手

这次的发布当然包括上周的内容，然后也说说一些复现的东西和想法。

上周做了个论文展示，用的是 revealjs。 不过官网上对 MarkDown 格式制作的方法并不推荐， 我转而找到了 这样一份文档。 除此以外，Quarto 工具底层也是调用 revealjs 的， 以后可以参考他们的调用方法。

前两天看到一个流体走迷宫的视频。 作为视频来说不一定做得特别好，但是这种题材几乎一看就是来自 YouTube 而不是 Bilibili。 中国创作者不偏好某种手工活，这只是种主观感觉而没有经过调研。

关于世界变得平均的观察， 这说明人类（在当今科技水平下）有一种普适的美学， 或许给每个人同样的足够的财富，大家住的都会是类似的房子，想想这种世界也挺无趣的？

Ciechanowski 老师的新作，经典的从初中知识点讲起。 对自行车的稳定性稍有涉及，不过最长见识的还是轮子之类的工程设计。

web 应用入门，不过不一定有意思。

科普 css 的 3D 相关参数， 但是比较有意思的是他的网页。看了个 V8 的科普文章也是用动画形式做的。

上周末试图在本地复现一个实时光追引擎， 它在 HN 上不知道为什么得到了几百个 upvote，我就去拉取试了一下，还帮作者测试了一下兼容性。 目前来说还没到看底层的程度，不过也可能去看看。 其实我更想知道其他实用引擎的发展现状。

除此以外，以前提到的 Word As Image 艺术字生成工具总算放出代码了，我也花了很长时间调试。 因为运行在 CPU 上，所以速度相当缓慢，而且试了几个字效果也不怎么样。挺好奇他们的示例是怎么做到的。

这周看到的一个音乐可视化小作品。

fast.ai 的复现 Stable Diffusion 教程， 有时间打算看看。希望能学到点东西。
